Adobe DreamWeaver 

O que é e para que serve o DreamWeaver? 

é um programa de design web que permite criar e gerir sites e aplicações web,ele combina uma interface visual e editor de códigos.

Como trabalhamos com o DreamWeaver? 

Podemos criar sites, páginas da internet, montar lojas virtuais,plataformas de notícia e outros diversos tipos de conteúdos digitais para colocar na Internet.

temos duas opções de usar o DreamWeaver, podemos criar os conteúdos dos sites incorporando os elementos por itens de Menus (comandos) de forma visual, ou criarmos o site programado (TAGs) com códigos em HTML, CSS e outras linguagens de programação Web.
